<Steps>
    <Step title="Configure Preferences">
        Configure your desired Zappl behaviour and user experience using the Zappl preferences configuration profile.
    </Step>
    <Step title="Install Supporting Profiles">
        Whitelist Zappl binaries using a managed login item configuration profile and a PPPC profile to allow Zappl to run seamlessly in the background.
    </Step>
    <Step title="Install Zappl">
        Deploy the latest Zappl installer using our installer script.
    </Step>
    <Step title="Profit.">
        You can now sit back and watch your compliance levels improve, with your Zappl reporting dashboard.
    </Step>
</Steps>
