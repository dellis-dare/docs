title: 'macOS Components'
icon: 'apple'
description: 'Zappl consists of key Agents and Launch Daemons.'
---

<Tabs>
  <Tab title="Agents">
    ```
    /Library/Application Support/DARE/Zappl
    ```
    >This is the root install directory for Zappl.
    <br />
    
    ```
    /usr/local/bin/zappl
    ```
    >This is a symlink which points to the main Zappl agent located at `/Library/Application Support/DARE/Zappl/Bin/zappl`. This agent is responsible for all software installs and updates, as well as sending reporting information to the Zappl portal.
    <br />
    
    ```
    /Library/Application Support/DARE/Zappl/Bin/uninstall
    ```
    >This agent can be used to uninstall Zappl.
    <br />
    
    ```
    /Library/Application Support/DARE/Zappl/Bin/updatemonitor
    ```
    >This agent kills the Zappl agent if it hangs during software updates/installs and sends error reports to the Zappl portal when this occurs.
  </Tab>
  <Tab title="Launch Daemons">
    ```
    /Library/LaunchDaemons/com.dare.zappl.run.plist
    ```
    >This daemon is responsible for running the Zappl agent on a recurring interval.
    <br />
    
    ```
    /Library/LaunchDaemons/com.dare.zappl.updatemonitor.plist
    ```
    >This daemon is responsible for running the Zappl update monitor agent whenever the Zappl agent initialises any updates/installs.
    </Tab>
</Tabs>
